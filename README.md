**Readme for Employee Time off calculator:**




These python files work together to calculate how long users in the "employees_list.csv" file have been on vacation.

Using "./exec_script.sh" via CLI to start, several questions will be asked, like:
* What is your name?
* When is the last time you clocked into work?"
* What is your department?"

<img width="553" height="162" alt="image" src="https://github.com/user-attachments/assets/e240a96e-fd8f-429d-ad9a-f7a50d0a6910" />




After that, the program will determine if an user is either ACTIVE, (UNDER) REVIEW, or DISABLED.

Once the program determines which category the user is in, the final result will be auto-exported to a pre-determined folder for backup.
